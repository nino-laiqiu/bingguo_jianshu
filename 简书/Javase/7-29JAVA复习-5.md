1.do while循环
**格式:**
    初始化语句;
     do {
           循环体语句;
           控制条件语句;
      } while(判断条件语句);


2.三种循环的区别
for 已知循环次数
while 已知总数 求次数的操作 先判断后执行
do while 循环至少循环一次 先执行后判断

4.跳转流程控制的三个关键字
break 跳出当前循环,执行循环外的代码.只能在switch和循环语句中使用,离开这两个会直接报错
continue 跳出本次循环 继续下一次循环 只能在循环中使用 离开场景就报错
return 结束正在运行的代码 相当于结束掉整个程序的运行(之后的都不运行) 没有特定场景还可以作为方法的返回值关键字使用
