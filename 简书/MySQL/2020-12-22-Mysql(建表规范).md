##1.hive建表的模板

```
create table dw.event_overview_aggr_user(
   guid     string,
   event_id  string, 
   act_cnt   int 
)
PARTITIONED BY (dy string) -- 分区
row format delimited fields terminated by ',' -- 行列分隔符
collection items terminated by '_' --MAP STRUCT 和 ARRAY 的分隔符(数据分割符号)
map keys terminated by ':' -- map中key和value的分隔符
lines terminated by '\n'; -- 行分隔符
STORED AS PARQUET -- 储存格式
TBLPROPERTIES("parquet.compress"="snappy") -- 压缩格式
```

##mysql中主键的设置的方法

给uid设置主键
```
CREATE TABLE user(
   uid INT PRIMARY KEY,
   uname VARCHAR(10),
   address VARCHAR(20)
)
```
给uid设置主键
```
CREATE TABLE user(
   uid INT,
   uname VARCHAR(10),
   address VARCHAR(20),
   PRIMARY KEY(uid)
)
```

设置uid的主键
```
CREATE TABLE user(
 uid INT,
 uname VARCHAR(10),
 address VARCHAR(20)
)
ALTER TABLE user ADD PRIMARY KEY(uid);
```

设置联合主键
```
CREATE TABLE user(
	uid INT,
	uname VARCHAR(10),
	address VARCHAR(20),
	PRIMARY KEY(uid,uname)
)
```
删除主键约束
```
ALTER TABLE USER DROP PRIMARY KEY;
```
