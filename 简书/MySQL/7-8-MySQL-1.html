7.8 MySQL-1

1.数据库服务器是不是MySQL?

对于数据库存储数据来说，国际上有mysql标准.很多软件开发商遵循SQL标准来开发一套软件，用来储存和管理数据，这套软件就可以称为数据库管理系统。

2.命令行黑窗口和mySQL服务器是什么关系？

客户端（client）和服务端（server）的关系就好像你的浏览器和163网站的服务器的关系一样

3.入门语句

连接# mysql -u username -ppassword

查看库#show databases

使用库#use databasename

查看表#show tables

创建库#create database 数据库名（charset 字符集）

删除数据库#drop database databasename

简单的建表语句#create table 表名（）

删除表# drop table 表名

给表改名#rename table 旧表名 to 新表名

清空表中数据#truncate 表名（truncate相当于重新建立一张新表 delete则是删除数据）

查看表中数据#select * from 表名

表中数据乱码#set names gbk/utf8
