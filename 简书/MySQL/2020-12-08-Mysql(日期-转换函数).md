>mysql hive中日期函数/转换函数的


**UNIX_TIMESTAMP()**
mysql中
select  UNIX_TIMESTAMP('2020-2-02 20:21:09')
hive中
select  UNIX_TIMESTAMP('2020-2-02 20:21:09','yyyy-MM-dd HH:mm:ss')

**date_format**
mysql中
select  date_format('2020-2-01 20:20:20' ,'%Y-%m-%d');
hive中
select  date_format('2020-2-01 20:20:20' ,'yyyy-MM-dd HH:mm:ss');

**substr**
SELECT substr('Spark SQL', 5);# k SQL 
SELECT substr('Spark SQL', 5, 1);# k
SELECT substr('Spark SQL', -3,2);# SQ

时间转成时间戳 时间戳转成时间
select UNIX_TIMESTAMP(now());
select  FROM_UNIXTIME(2008621146);
