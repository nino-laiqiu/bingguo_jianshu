-1.cd 到某目录
-2.ls -a ls -l  -i(列出每个文件的inode)
-3.pwd  显示当前位置
-4.mkdir 创建文件夹 mkdir -p 递归创建文件夹
-5.touch  创建文件
-6.echo 打印 覆盖文件内容 追加文件内容
>echo "hello world"打印
echo > 1.txt 覆盖操作
echo >> 1.txt 追加操作

-7.rm 删除 rm-r 询问递归删除  rm -rf 强制删除
-8.cp 复制粘贴   cp  1.txt  (目录/文件)   目录则是复制到  文件存在则是覆盖 不存在生成一个内容一样的
-9.mv  1.txt 2.txt  第二个存在则是覆盖 ，不存在则是重命名
-10.cat 查看文件内容 cat -A 查看内容(包括隐藏)  cat -n 查看显示行号
-11.head 查看文件内容(默认前10行) head -n  查看n行

注意:less  find  grep 感觉很重要!!!!!!

-12.tail 查看文件后几行
-13.in ln -s   [源文件]    [目标文件]    创建软链接       源文件 使用 绝对路径
  ln     [源文件]    [目标文件]    创建硬链接
-14. which  查看系统命令所在目录
-15. whereis  搜索命令所在目录 配置文件所在目录  及帮助文档路径      
-16.grep  在文件中搜寻字串匹配的行并输出 在命令的输出中搜索内容     
-17.find [搜索路径] [匹配条件]
-name “文件名” (支持通配符)
-type   类型
 f 二进制文件  l 软连接文件 d 目录  c 字符文件 
-size  按文件大小查找
18.tar  
tar -zcvf dir1.tar.gz  dir1   使用gzip将目录dir1压缩成一个打包并压缩文件
tar -zxvf  dir1.tar.gz  -C(大写的)  /root     解压到指定的/root目录
19.重启网络服务 service network start
20.Linux所提供的管道符“|”将两个命令隔开，管道符左边命令的输出就会作为管道符右边命令的输入。连续使用管道意味着第一个命令的输出会作为第二个命令的输入，第二个命令的输出又会作为第三个命令的输入，依此类推。
