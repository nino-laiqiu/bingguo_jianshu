1.上传解压
rz   hbase-2.2.5-bin.tar.gz
tar -zxf  hbase-2.2.5-bin.tar.gz
2.时间同步
yum install -y ntpdate
ntpdate 0.asia.pool.ntp.org 
3.配置修改
>/opt/hbase/hbase-2.2.5/conf/

![image.png](https://upload-images.jianshu.io/upload_images/9049859-ada9109c035d077e.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

>vi hbase-env.sh 修改内容：注意放开注释
```
export JAVA_HOME= /opt/java/jdk1.8.0_261/
export HBASE_MANAGES_ZK=false
```

>vi hbase-site.xml
```

<!-- 指定hbase在HDFS上存储的路径 -->
<property>
<name>hbase.rootdir</name>
<value>hdfs://linux03:8020/hbase</value>
</property>
<!-- 指定hbase是分布式的 -->
<property>
<name>hbase.cluster.distributed</name>
<value>true</value>
</property>
<property>
  <name>hbase.unsafe.stream.capability.enforce</name>
  <value>false</value>
</property>
<!-- 指定zk的地址，多个用“,”分割 -->
<property>
<name>hbase.zookeeper.quorum</name>
<value>linux03:2181,linux04:2181,linux05:2181</value>
</property>
```

> regionservers 配置  启动集群中的Regionserver机器

```
linux03
linux04
linux05
```

4.分发

>scp -r /opt/hbase/ linux04:$PWD

>scp -r /opt/hbase/ linux05:$PWD

修改环境变量
```
 export 	HBASE_HOME=/opt/hbase/hbase-2.2.5
 export  PATH=$PATH:$JAVA_HOME/bin:$TOMCAT_HOME/bin:$HADOOP_HOME/bin:$HADOOP_HOME/sbin:$ZOOKEEPER_HOME/bin:$HBASE_HOME/bin
```
5.启动命令与一键启动

  >bin/hbase-daemon.sh start master

 > bin/hbase-daemon.sh start regionserver

>  bin/start-hbase.sh   一键启动

6.进入HBASE

> bin/hbase  shell
