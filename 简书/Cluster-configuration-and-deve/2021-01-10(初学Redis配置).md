##1.wget下载
> wget http://download.redis.io/releases/redis-6.0.8.tar.gz

##2.解压与安装gcc语言环境
> tar -zxf redis-6.0.8.tar.gz 

> yum install gcc-c++

##3.编译Redis
> make

##4.出错的解决
> make  distclean清除
升级GCC环境为9版本
> yum  -y  install  centos-release-scl
> yum  -y  install  devtoolset-9-gcc  devtoolset-9-gcc-c++  devtoolset-9-binutils

> scl enable devtoolset-9 bash 环境配置暂时有效
> echo "source /opt/rh/devtoolset-9/enable" >>/etc/profile 环境配置(注意是自己的目录)

##5.接着重新编译和安装
> make
> make  install  不指定安装位置   将Redis安装在Linux默认位置：/usr/local/bin
> make  install  PREFIX=/usr/local/redis-6.x　　　　　将Redis安装在指定位置：　/usr/local/redis-6.x/bin   （bin是安装自动生成的） 指定安装位置
 
##6.配置Redis
![image.png](https://upload-images.jianshu.io/upload_images/9049859-9e197bb3196b2fb3.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

把 redis.conf 复制到Redis目录下

> cp /opt/redis/redis-6.0.8/redis.conf  /usr/local/bin/ (注意是自己的目录)

##7.启动服务器和客户端
> ./redis.server redis.conf 
>  ./redis.server redis.conf  & 在后台运行

> ./redis-cli
