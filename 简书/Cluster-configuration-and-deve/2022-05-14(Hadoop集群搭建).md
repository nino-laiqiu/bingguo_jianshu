##1. 虚拟机准备
#####打开VM修改，修改子网IP和NAT设置，要在同一段
![打开VM修改，修改子网IP和NAT设置，要在同一段](https://upload-images.jianshu.io/upload_images/9049859-82f5a1e22d6418ee.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

#####更改VMnet8设置
![更改VMnet8设置](https://upload-images.jianshu.io/upload_images/9049859-13fe1bce854e85e4.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)


#####vi   /etc/sysconfig/network-scripts/ifcfg-ens33  修改配置
```
BOOTPROTO =static
ON\NBOOT =yes
UUID 删除
IPADDR =IP地址
GATEWAY =网关
NETMASK =子网掩码
DNS1 =(可设置多个)
```
![/etc/sysconfig/network-scripts/ifcfg-ens33  修改配置](https://upload-images.jianshu.io/upload_images/9049859-bfb06e9918201081.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240)

这里切记：ONBOOT要设置为yes ,BOOTPROTO =static,我这里ONBOOT没有设置导致没有ping通

#####配置主机名称和IP映射
主机名称 /etc/hostname
