##SQL字符串函数
#####进制函数
```
SELECT BIN(12); 
```
转为二进制,这个函数在bitmap处理那块用到过
```
SELECT CONV(16,10,2)
```
碱基间转换数字,如上将10进制的转化为2进制的数,这个函数也在bitmap那块使用过


#####两个常用字符串拼接函数
```
SELECT CONCAT_WS(',','携','程' ); 
```
```
SELECT CONCAT('携','程' );
```

#####字符截取函数


INSERT 字符替换,pos是位置,len是替换的个数,例如下
```
SELECT INSERT('abcdefg', 3, 4, 'qwzq')
```

INSTR  返回匹配字符串,匹配的初始索引
```
SELECT INSTR('foobarbar', 'bar');
```
left函数,从左截取字符个数,right函数,从右截取字符个数,`substr`函数既能从左边截取又能从右边截取
```
SELECT LEFT('foobarbar', 5);
```
lpad函数,补充左边空余位,rpad是补充右边空余位置
```
LPAD（STR，LEN，padstr填补）
```

#####字符串函数split、array_contain
Hive用array_contains判断列表是否包含某个元素,例如:
```
select if(array_CONTAINS(array(0,1),0) = true ,1,0)
```
split支持多个分隔符
```
SELECT split('hello world,zhuoru',' |,')
```
判断与过滤 
```
array_CONTAINS( split( other ,'\002' ) ,'leakscan' || '\003' || '-' ) = TRUE
```
注意这个函数的使用场景(array_contains)
#####替换
replace函数替换空格,例如
```
select replace(' 1 , 21 ,2 , 3 ','\002|\003','') 
```
等价于剔除空格
```
select TRIM(' 1 , 21 ,2 , 3 ')
```
替换
```
SELECT REPLACE('www.mysql.com', 'w', 'Ww')
````
