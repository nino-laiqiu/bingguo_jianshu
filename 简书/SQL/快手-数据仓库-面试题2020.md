##1.用户登陆日志
app, user_id, datetime
a1,user1,2020-01-01 00:00:00
a2,user2,2020-01-01 00:00:01
a3,user3,2020-01-01 00:00:02
a4,user4,2020-01-01 00:00:03
……
输出每个app下访问次数最多的5个用户

```
with x as (
select
app,user_id,count(1) as rn
from
table1 
group by app,user_id
)
select 
t.app,
t.user_id
from 
(
select
app,
user_id,
row_number() over(partition by app order by rn desc) as max_user
from 
x
) as t 
where t.max_user <= 5
```

##2.认识的组合数
现有城市网吧访问数据，字段：网吧id，访客id（身份证号），上线时间，下线时间
规则1、如果有两个用户在一家网吧的前后上下线时间在10分钟以内，则两人可能认识
规则2、如果这两个用户在三家以上网吧出现【规则1】的情况，则两人一定认识
需求：
该城市上网用户中两人一定认识的组合数

//自连接
//最初是限制了t0.uid!=t1.uid，这是筛选出同一个网吧不同id的组合，但是这样会有个问题，A-B与B-A实际上是一个意思，却出现了两条。我们只需要一条数据就可以了，能说明A与B是认识的。在这里是取B-A。或者t0.uid<t1.uid也可以的
```
设表名：table0
字段：wid ， uid ，ontime ，offtime
select
	id,
	count(distinct wid) c
from
	(select
		wid,
		concat(t0.uid,t1.uid) as id
	from
		(select
			wid,
			uid,
			unix_timestamp(ontime,'yyyyMMdd HH:mm:ss') as ontime,
			unix_timestamp(offtime,'yyyyMMdd HH:mm:ss') as offtime
		from 
			table0
		)t0
	join
		(select
			wid,
			uid,
			unix_timestamp(ontime,'yyyyMMdd HH:mm:ss') as ontime,
			unix_timestamp(offtime,'yyyyMMdd HH:mm:ss') as offtime
		from 
			table0
		)t1
	on t0.wid=t1.wid
	    and t0.uid>t1.uid
		and (abs(t0.ontime-t1.ontime)<10*60 or abs(t0.offtime-t1.offtime)<10*60)
		
	)t0
group by 
	id
having
	c>=3
```
方法二,来自王知无大数据(保留答案,这里有个问题就是出现在不同网吧的可能不是同两个人,这里应该要lag用户id进行concat拼接,这里的难点就是排序了,可以在窗口中预先按照uid来排序来避免这个问题)
```
select count(uid) as com_cnt
from(
    select uid
          ,count(1) as flag
    from(
        select wid
              ,uid
              ,abs(unix_timestamp(ontime,'yyyy-MM-dd HH:mm:ss')-unix_timestamp(lag(ontime,1,'1970-01-01 08:00:00') over(partition by wid order by ontime),'yyyy-MM-dd HH:mm:ss')) / 60 ontime_diff
              ,abs(unix_timestamp(offtime,'yyyy-MM-dd HH:mm:ss')-unix_timestamp(lag(offtime,1,'1970-01-01 08:00:00') over(partition by wid order by offtime),'yyyy-MM-dd HH:mm:ss')) / 60 offtime_diff
        from table0
    ) m
    where ontime_diff<=10 and offtime_diff<=10
    group by uid
) n
where flag>=3
```
##3.收入前一万
现在有中国人从1980年到2020年每年每人的收入记录表A如下：
id，year，income
求每年收入前一万的人

```
with x as (
select
id,year,income,
row_number() over(partition by year ,substr(income,1,4) order by income desc ) as rn  
from 
table3
where rn <= 10000
) 
select 
t.id,
t.year
from 
(
select
id,year，income,
row_number() over(partition by year order by income desc ) as rm 
from 
x 
where rm <= 10000
) t
```
