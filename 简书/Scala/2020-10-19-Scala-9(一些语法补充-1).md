#####5.条件选择表达式match
与java switch不同之处:
1.scala的case支持任何类型,java则不是
2.scala每个分支后不需要写break,在case语句中默认就有
3.scala中match语句是有返回值的

```
  def main(args: Array[String]): Unit = {
    val strings = Array("A", "B", "C")
    for (elem <- strings) {
    var result =  elem match {
        case "A"=>println("我是A")
        case "B"=>println("我是B")
        case "C"=>println("我是C")
      }
      println(result ) //（）
    }
  }
}
```
