##4.继承与特质
##### 注意事项(在子类中重写了父类的val变量)
```
object Test1 {


  def main(args: Array[String]): Unit = {
    println("===")
    val employee = new Employee
    employee.array.mkString(",").foreach(println)  
    //父类构造器
    //子类构造器
    //0
  }
}

class  Person{
        println("父类构造器")
        val num :Int = 10
        val array :Array[Int] =Array[Int](num)
}

class Employee extends  Person{
      println("子类构造器")
      override val num: Int = 2
}
```
(对该现象的解释,首先调用父类的构造器初始化,执行array 要寻找num 但是num被重写了,找寻子类的num但是子类构造器还没有初始化)

解决的方法:
1.将变量用final关键字修饰
2.将变量用lazy 修饰 即懒加载

##5,叠加特质的构造与执行顺序
构造顺序从左往右,有父类先构造父类,
执行顺序,从右往左,有父类执行父类,注意这里的父类只的是叠加顺序的左边的类

##6.闭包
定义:
在创建函数时,如果需要捕获自由变量,那么包含指向被捕获变量的引用的函数称为闭包函数
```
object Test2 {
  def main(args: Array[String]): Unit = {
    println(add(1)(8))
   }

   def add(x:Int)={
     (y:Int) => {
           x+y
     }
   }
  
  def  add1(x:Int) = (y:Int) => x+y
}
```
